API para la aplicación Web: Tablero de Visualización de Anomalías

  • POST /cargas + datos + usuario - Sube registros a la base de datos
  • POST /tableros + configuración + usuario - Guarda un tablero a la base de datos
  • GET /tableros + id + usuario - Consulta uno de los tableros guardados
  • DELETE /tableros + id + usuario - Borra uno de los tableros guardados
