API para la aplicación Web: Tablero de Visualización de Anomalías

  • GET / - Ventana actual con instrucciones del API
  • GET /datos-disponibles + id_usuario - Consulta las cargas y tableros de este usuario
  • POST /cargas + datos + columnas + id_usuario - Suben registros a la base de datos
  • GET /cargas + id_carga + id_usuario - Se consultan los registros de la base de datos
  • DELETE /cargas + id_carga + id_usuario - Se borra una carga de datos con sus tableros asociados
  • POST /tableros + configuración + id_carga + id_usuario - Guarda una configuración de tablero a la base de datos
  • GET /tableros + id_tablero + id_usuario - Consulta uno de los tableros guardados
  • DELETE /tableros + id_tablero + id_usuario - Se borra un tablero de datos, su configuración
